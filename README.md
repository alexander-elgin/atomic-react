# React Redux PWA boilerplate

A front-end boilerplate based on React & Redux

## Main Features
- [atomic design](http://bradfrost.com/blog/post/atomic-web-design/)
- ES6
- eslint
- i18N translation
- jest
- material-ui
- postcss
- react-icons
- redux-form
- redux-saga
- reselect
- storybook
- styled-components
- token-based authentication
- webpack

## Dependencies
- Node.js
- npm OR yarn
- for auth features (back end) you can use [express-passport-boilerplate](https://github.com/alexander-elgin/express-passport-boilerplate)

## Install
```sh
$ yarn install
```

## Launch
```sh
$ yarn start
```

## Tests
```sh
$ yarn test
```
